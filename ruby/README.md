# xstream.rb

Public key encryption system combining X25519 Diffie-Hellman with the STREAM construction

## Installation

Add this line to your application's Gemfile:

```ruby
gem "xstream"
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install xstream
